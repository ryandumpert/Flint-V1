## üéì **LESSONS LEARNED:**

### **Template Prop Rules:**

1. **MediaText layouts:** Always use camelCase
   - ‚úÖ `"imageLeft"`, `"imageRight"`, `"twoColumn"`
   - ‚ùå `"text-left"`, `"image-right"`, `"two-column"`

2. **Lesson content:** Always use `sections` array
   - ‚úÖ `"sections": [{ "title": "...", "content": "..." }]`
   - ‚ùå `"content": "..."`

3. **Compare columns:** Always use `columns` with `variant`
   - ‚úÖ `"columns": [{ "title": "...", "items": [...], "variant": "bad" }]`
   - ‚ùå `"leftTitle"`, `"rightTitle"`, `"rows"`
