# Add Knowledge Workflow

Add new domain knowledge to `public/prompts/tele-knowledge.md`.

## What Goes Here

- Facts about Mobeus
- The Launch Event details
- Brand messaging
- Real examples and stories
- Impact statistics

## Format

Keep it concise. Use markdown:

```markdown
## SECTION NAME

Brief description of the topic.

**Key points:**
- Point 1
- Point 2
- Point 3
```

## After Adding

Run `/publish` to deploy.
